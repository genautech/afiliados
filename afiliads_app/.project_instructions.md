# AfiliAds — Central de Campanhas

## Overview
App completo para afiliados digitais que operam no ClickBank, BuyGoods, MaxWeb, Hotmart, Eduzz e Monetizze com Google Ads.

## Tech Stack
- Dark theme (bg #0f172a, cards #1e293b, borders #334155)
- Cores: verde #22c55e (scale/sucesso), vermelho #ef4444 (kill), amarelo #f59e0b (otimizar), azul #3b82f6 (ativo)
- Auth: email/password com NextAuth + Prisma adapter
- LLM: gpt-5.4-mini para gerador RSA com streaming
- Charts: Recharts para gráficos diários

## Pages Structure
- /login, /signup — Auth pages
- /dashboard — Métricas consolidadas, alertas, cards por plataforma
- /wizard — Wizard 9 passos para nova campanha (oferta → break-even → anti-strike → bridge → keywords → naming → google ads → tracking → go-live)
- /campanhas — Gerenciador com filtros, decisões KILL/OTIMIZAR/SCALE
- /diario — Registro diário + gráficos EPC/CPC com linha de break-even
- /keywords — Pesquisa, sugestões por vertical/camada, negativas, Answer The Public link
- /rsa — Gerador RSA com IA (streaming), validação de claims proibidos
- /conhecimento — Base de conhecimento com tabs (Estratégias, Google Ads, Playbooks, Templates, Checklists) + sidebar de erros comuns
- /configuracoes — Credenciais das plataformas (mascaradas após salvar)

## Database Tables
campaigns, campaign_checklists, keywords, daily_logs, campaign_decisions, integrations, knowledge_base_searches

## Key Decisions
- Answer The Public: bloqueia iframe, substituído por link direto em nova aba
- Wizard salva progresso a cada passo via API
- App funciona 100% sem APIs configuradas (dados manuais)
- Conteúdo do SKILL.md hardcoded no componente de conhecimento
- Idioma: PT-BR com termos técnicos em inglês (EPC, CPC, RSA, UTM)
