# Nomenclatura Padrão de Campanhas

## Formato

```
[TIPO]-[NICHO]-[OBJETIVO]-[FASE]           ← campanha
[CAMPANHA] > [TEMA-DO-GRUPO]               ← grupo de anúncios
```

- **TIPO:** PESQ (Pesquisa) · PMAX · YT (YouTube) · RMKT (remarketing)
- **NICHO:** abreviação fixa do nicho/produto (ex.: INGLES, IA-TOOLS, PROTOCOLO)
- **OBJETIVO:** CONVERSAO · LEADS · TRAFEGO
- **FASE:** VALIDACAO · OTIMIZACAO · ESCALA
- Para internacional, prefixe o país: `US-PESQ-AITOOLS-CONVERSAO-VALIDACAO`

## Exemplos

```
PESQ-INGLES-CONVERSAO-VALIDACAO
PESQ-INGLES-CONVERSAO-VALIDACAO > curso de ingles online
PESQ-INGLES-CONVERSAO-VALIDACAO > aprender ingles sozinho
US-PESQ-AITOOLS-CONVERSAO-OTIMIZACAO > best ai writing tool
RMKT-PROTOCOLO-CONVERSAO-ESCALA > visitantes pre-venda 30d
```

## Regras

1. A FASE muda no nome quando a campanha muda de fase (duplicar/renomear mantém histórico legível)
2. Um tema por grupo; o nome do grupo = a keyword principal dele
3. Nunca misturar países na mesma campanha — o prefixo de país é obrigatório fora do BR
4. Listas de negativas compartilhadas: `NEG-MASTER-[NICHO]` e `NEG-GERAL`
5. Conversões: `VENDA-[PLATAFORMA]` (primária) e `CLIQUE-CTA-[NICHO]` (secundária)
