# Skills — ponte para o que já existe

Não duplique skills. O Hermes (e o Cursor) devem apontar para os arquivos canônicos:

| Skill | Path | Quando usar |
| --- | --- | --- |
| Estratégias multi-rede | `/SKILL.md` | ClickBank, BuyGoods, MaxWeb, Hotmart + Google Ads |
| Afiliado Google Ads Pro | `/afiliado-google-ads-pro/SKILL.md` | Campanha, RSA, presell, tracking, kill/scale |
| Referências | `/afiliado-google-ads-pro/references/` | Carregar só o arquivo do fluxo |
| Scripts | `/afiliado-google-ads-pro/scripts/` | Métricas, planilha, validar copy, relatório |

Skills `.skill` em `Uploads/` são pacotes; preferir as pastas versionadas acima.

## Regra

Se uma rotina for previsível → skill ou script.  
Se precisar de julgamento sobre dados novos → agente Hermes com skill carregada.  
Se for só conta/álgebra → código, nunca LLM.
